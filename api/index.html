Análisis de sentimientos — IMDB Movie Reviews

Laboratorio PLN

Se entrenaron y compararon 7 modelos para clasificar reseñas de películas como positivas o negativas. El criterio principal de evaluación fue el F1 Score (weighted).

¿Por qué el baseline fue el mejor modelo?

El modelo TF-IDF + Regresión Logística obtuvo el mejor F1 Score entre todos los experimentos. Esto se explica porque el dataset IMDB en español tiene un vocabulario relativamente estable y las reseñas contienen palabras clave muy discriminativas (como "excelente", "terrible", "aburrida"). TF-IDF captura exactamente estas señales léxicas sin necesidad de representaciones semánticas complejas. Las redes neuronales con embeddings requieren más datos y más épocas para superar esta ventaja en tareas donde el léxico es suficientemente informativo.
